Citações

Uma citação é um trecho de texto que foi escrita ou dita por outra pessoa.
Exemplo:
Segundo pai dos burros:
Semântica é o significado dos vocábulos, por oposiçaõ á sua forma.

Para criar uma citação em HTML, podemos usar a tag <q> que vai receber automaticamente as aspas, mas não terá nenhum deslocamento.
É mais usada quando queremos fazer uma citação no meio de um parágrafo(semântico)

Citação mais longa

Também podemos criar citações mais longas (Em blocos) e que tenham um parágrafo só para si. N
esse caso, colocamos tudo dentro de <blockquote>. O texto ganha um recuo automaticamente.
Podemos adicionar um link da citação original usando o parâmetro cite dentre da tag.
Exemplo:
Frase de Albert Einstein:
Deus não escolhe os capacitados, capacita os escolhidos. Fazer ou não fazer algo só depende de nossa vontade e perseverança.
